环境搭建

项目结构搭建

  • parent模块仅仅用来确定各个Maven依赖的半杯
  • webui、component、entity模块继承parent模块
  • uti、reverse模块属于独立工程，不参与继承与聚合
  • webui依赖component。
  • component依赖entity、util。

创建空工程

  1. 新建一个空项目，命名为 atcrowdfunding
  2. 新建三个 Maven Module，不选择模板(即不勾选 Create from archetype)
    • admin-parent
    • common-util
    • common-reverse
  3. 新建三个 Maven Module，parent的子
    • admin-webui
    • admin-component
    • admin-entity
  4. 选择打包方式
    • admin-parent为pom
    • admin-webui为war
    • 其它Module下的jar

数据库创建

基于Maven的MyBatis逆向工程

  1. 逆向工程为reverse模块

  2. 配置Maven（pom.xml）

  3. 在Resources目录下新建文件 generatorConfig.xml

运行mybatis-generator:generate之后idea会帮我们创建相应的类的mapper文件

会在指定目录的entity包下生成Admin.java、AdminExample.java，在mapper包下生成AdminMapper.java、AdminMapper.xml，在Admin.java补充有参无参构造器+tostring方法

将生成的逆向工程文件放到子工程中：cn.wudou.crowd.entity包整体移动到entity子工程下；component子工程src/main/java目录下创建cn.wudou.crowd.mapper包，将AdminMapper.java移过来；webui子工程src/main/resources目录下创建mybatis.mapper包，将AdminMapper.xml移过来
